CX-5461 Immunogenomics Workflow

A framework for systematic neoantigen prediction and biomarker discovery in HRD-high breast cancer, scalable to colorectal cancer (CRC) applications.

Breast Cancer Optimized Scalable to CRC

Overview

This project demonstrates a 4-step computational pipeline that models neoantigen generation following CX-5461 induced stress. By integrating public data (TCGA, COSMIC) with experimental signatures (GSE), the framework identifies conserved biomarkers and high-affinity epitopes for chemo-immunotherapy combination strategies.

Phase 01

Data Acquisition

Retrieval of TCGA-BRCA cohorts (N~1,100) using GDC API, focused on HRD-high populations sensitive to G-quadruplex stabilizers.

GDCquery(
  project = "TCGA-BRCA",
  data.category = "Transcriptome"
)
Phase 02

Predictive Modeling

Simulation of collateral mutagenesis using the SBS-CX-5461 signature. Prediction of MHC binding via NetMHCpan-4.1.

netMHCpan -p peptides.txt \
-a HLA-A*02:01 -BA
Phase 03

Multi-Omics Integration

Correlating predicted epitopes with experimental CX-5461 RNA-seq data to prioritize upregulated, high-affinity candidates.

Phase 04

Deep Learning

Ensemble ML approach (NeoPredictor) integrating peptide sequences and HRD scores for immunogenicity ranking.
